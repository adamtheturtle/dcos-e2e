Usage: minidcos vagrant destroy-list [OPTIONS] [CLUSTER_IDS]...

  Destroy clusters.

  To destroy all clusters, run ``minidcos vagrant destroy $(minidcos vagrant
  list)``.

Options:
  --enable-spinner / --no-enable-spinner
                                  Whether to show a spinner animation. This
                                  defaults to true if stdout is a TTY.
  -h, --help                      Show this message and exit.
